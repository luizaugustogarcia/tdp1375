TDP 1.375

This document proves that if $\norm{\spi}_3 \geq 8$, then a $\frac{11}{8}$-sequence exists. In order to understand the details of the proof, the reader is referred to the paper by L. A. G. Silva, L. A. B. Kowada, N. R. Rocco and M. E. M. T. Walter. The syntax of the proof is explained here.

Results

Lemma 1: Every sufficient configuration $\Gamma$ of $\spi$ has a $\frac{11}{8}$-sequence of transpositions.

Lemma 2: The bad small components that do not have a $\frac{11}{8}$-sequence are:

  1. the bad oriented $5$-cycle
  2. the unoriented interleaving pair
  3. the unoriented necklace of size $4$
  4. the twisted necklace of size $4$
  5. the unoriented necklace of size $5$
  6. the unoriented necklace of size $6$
The demonstration of the previous results rely on a case analysis performed similarly to a DFS search, in which each configuration is systematically extended until $\frac{11}{8}$-sequences are found, starting from:
  1. the bad oriented $5$-cycle
  2. the unoriented interleaving pair
  3. the unoriented intersecting pair
No configuration with $3$-norm equals to $10$ was reached by the search. The search found the full configuration (0 5 2)(1 10 3)(4 9 7)(6 11 8) which does not allow a $\frac{11}{8}$-sequence. However, note that this configuration is not valid, since in this case, $\esigma\epi\esigma^{-1}$ ($\epi$ conjugated by $\esigma$) is not a $(n+1)$-cycle. Intuitively, this means that there are no $\epi$ and $\esigma$ such that $\spi$ contains a component with the form of this invalid configuration.

Lemma 3: If $\spi$ consists only of bad small components and $\norm{\spi}_3 \geq 8$, then a $\frac{11}{8}$-sequence of transpositions exists.

In order to demonstrate this result, we also employ an approach similar to a DFS search, in which, starting from each bad small component listed above, we extend the previous configuration by adding another bad small component in a different position of $\epi$, until find a $\frac{11}{8}$-sequence. We note that by this approach no combination with $3$-norm bigger than 7 was extended. Below, the result of the search:
  1. the bad oriented $5$-cycle
  2. the unoriented interleaving pair
  3. the unoriented necklace of size $4$
  4. the twisted necklace of size $4$
  5. the unoriented necklace of size $5$
  6. the unoriented necklace of size $6$

Lemma 4: If there is an even (odd-length) $k$-cycle $\gamma=(a\dots b\dots c\dots)$ in $\spi$ such that $k\geq 7$ and $(a,b,c)$ is an oriented triple, then there is a $\frac{4}{3}$-sequence.

Proof. If $(a\;b\;c)$ is a $2$-move, then the lemma holds since a $2$-move is a $(1,1)$-sequence, which in turn is $\frac{4}{3}$-sequence. Thus, w.l.o.g, suppose \[\gamma=(\underbrace{d\;e\dots b}_{odd}\;|\;\underbrace{f\dots c}_{even}\;|\;\underbrace{g\dots a}_{even}\;|).\] We use vertical bars to indicate the positions in $\gamma$ that would be affected by the application of $(a\;b\;c)$ (breaking $\gamma$ into $3$ smaller cycles), and subscripts to indicate the length parity of the resulting cycles. Note that the cycle $\gamma$ can be rewritten as the product \[\gamma=(\underbrace{a\dots}_{odd})(\underbrace{b\dots}_{odd})(\underbrace{c\dots}_{odd})(a\;d\;e\;b\;f\;c\;g).\] For each of the $71$ distinct possible forms of $\epi$, related to the symbols of $(a\;d\;e\;b\;f\;c\;g)$, there is a $\frac{4}{3}$-sequence of transpositions.
